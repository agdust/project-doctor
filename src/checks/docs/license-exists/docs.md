# license-exists

Checks that a LICENSE file exists.

## Why

Without a license:

- Others legally cannot use your code
- Contributions are legally ambiguous
- Companies may avoid your project
- Open source platforms may flag it

## Examples

**Pass**: `LICENSE` or `LICENSE.md` file exists.

**Fail**: No license file found.

## How to fix

Choose a license at https://choosealicense.com/

Common options:
- **MIT** - Permissive, simple
- **Apache 2.0** - Permissive with patent protection
- **GPL 3.0** - Copyleft, derivatives must be open

Create `LICENSE` file with the full license text.
