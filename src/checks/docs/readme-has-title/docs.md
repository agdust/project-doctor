# readme-has-title

Checks that README.md starts with a title heading.

## Why

A title:

- Clearly identifies the project
- Displays properly on GitHub/npm
- Helps with SEO and discoverability
- Sets professional tone

## Examples

**Pass**: README.md starts with `# Project Name`.

**Fail**: README.md doesn't start with a heading.

**Skip**: No README.md file.

## How to fix

Start README.md with a level-1 heading:

```markdown
# Your Project Name

Rest of the readme...
```
