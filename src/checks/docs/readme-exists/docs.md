# readme-exists

Checks that a README.md file exists.

## Why

The README is often the first thing people see. It should:

- Explain what the project does
- Show how to install and use it
- Link to documentation
- Provide contribution guidelines

## Examples

**Pass**: `README.md` file exists in project root.

**Fail**: No `README.md` found.

## How to fix

Create a `README.md` with at minimum:

```markdown
# Project Name

Brief description of what this project does.

## Installation

```bash
npm install
```

## Usage

Basic usage example.
```
